# Panduan untuk Kontributor Knowledge Base Alterra

## Akun Kontributor

```
Kontributor resmi Knowledge Base Alterra dapat mengedit dokumentasi (format Google Docs) secara langsung. Jika kamu tertarik menjadi kontributor, silakan mengikuti cara di bawah ini.
```

Update Document

1. Konten di **Knowledge Base Alterra** menggunakan bahasa Inggris dan bahasa Indonesia

2. Format konten yang ada di **Knowledge Base Alterra** berbentuk *markdown*, pdf, dan Google Docs

3. Format dokumen **Google Docs** memiliki tombol "Edit" di bagian atas halaman 

   ![](/Users/shindy/Downloads/Getting-Started/images/Screen Shot 2020-05-12 at 05.54.09.png)  

   jika ingin mengedit konten, kamu dapat mengeklik tombol tersebut dan kirim *request permission*

4. *Request* akan diberikan dalam waktu paling lambat 2x24 jam

5. Setelah konten diubah, akan melalui *review* oleh Tim *Technical Writer*



Create New Document

1. Untuk mendapatkan akses `write` klik tombol "Create New" pada kanan atas halaman

2. Tombol tersebut akan *redirect* ke static-page dokumen baru

3. User tekan tombol edit dan open window Google Docs (butuh *request permission* ke Admin)

4. Jika butuh training untuk penggunaan **Knowledge Base Alterra** silahkan hubungi thoth@alterra.id
